3. Drags/drops or clicks to upload photo
4. Enters pet name, type, breed
5. Clicks upload

**How it works:**
```
User selects file
    ↓
Browser validates it
    ↓
Shows preview
    ↓
User enters details
    ↓
Form submits to API
    ↓
API validates session
    ↓
Saves file to disk
    ↓
Saves to database
    ↓
Responds with success
    ↓
Redirect to next page
```

## 🐛 Quick Troubleshooting

**Problem:** "Cannot find module react-dropzone"
**Fix:** Run `npm install` (should already be installed)

**Problem:** Upload fails with 401
**Fix:** Make sure you're logged in with Google OAuth

**Problem:** "pets" table not found error
**Fix:** Run the Supabase SQL from DAY3_PET_UPLOAD_CODE.ts FILE 4

**Problem:** File not saving to public/uploads
**Fix:** Check folder permissions, or create public/uploads manually

**Problem:** Page won't load
**Fix:** Check console errors, make sure components are imported correctly
